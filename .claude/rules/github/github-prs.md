# GitHub Pull Request Standards

## PR Template Structure

**MUST**: Include these sections in PR description:
- Summary (3-5 bullet points)
- Test Plan
- Changes (Added/Modified/Removed)
- Related Issues

## Title Format

**MUST**: Use format: `Issue #<number>: <Descriptive title>`

## Summary Requirements

**MUST**: Provide 3-5 bullet points summarizing the changes
